Nome do projeto: Bar da Miranda – Sistema de Comandas

Objetivo do sistema: Organizar o caixa e abrir comandas para que não corra risco de passar alguma conta sem pagar.

Escopo: Cadastro de produtos, Registro de comandas, Formas de pagamento, Cardápio, Relatório, Categorias dos produtos, Lista de pesquisa dos produtos e dos clientes

Não terá: Comandas repetidas, Controle de estoque

Funcionalidade principal: Ter controle do que se vende no dia, organização e fechamento de caixa.

Tecnologias principais: Banco de dados: SUPA base (ou Supabase), Cloud (nuvem)
